note:
1. normal : by default, played as normal(forwards)
2. reverse: played reverse direction (backwards)
3. alternate : played in forward(first), then backwards
4. alternate-reverse: played in backwards(first),then forwards

#Display Property : 
1. inline : just like<span> tag any height and width properties will not work 
2. block : just like <p> tag start on new line and takes up whole width 
3. inline-black : like inline block container you can apply height width property 
4. none: completely remove element
